# Тестовые потоки для домашнего задания

```shell
git clone https://github.com/testhubik/homework.git
cd homework
npm i
npm start
```

Откройте браузер на странице http://localhost:3102/

Ссылки на тестовые потоки можно увидеть в коде страницы.
